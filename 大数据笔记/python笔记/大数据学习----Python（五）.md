####数据类型的转换

把一种类型的数据转化为另一种类型，每种类型的构造函数都可以完成转换

**转为float类型**

用float（）函数

```python
float(10)        #10.0
float("12.8")    #12.8
float("4")       #4.0
```

**转为int类型**

```python
int(21.34)       #21
int("2")         #2
```

**round函数**

取值到离目标整数最近的数，比如3.49999离3最近，那么round(3.49999) = 3

而3.500001  那么round(3.500001) = 4

四舍五入

**转str**

```python
str(1.23)           #"1.23"
```
